# PMWeatherCM1

Addon that adds support of CM1 simulations for Protomanly's Weather Mod.

# Installation

Due to file size constraints, the scenarios (simulations) are hosted here.

Download the .jar file from Modrinth and PMWeatherEnhancements resource pack.

Download scenarios from Releases tab.

Place the .jar in mods folder.

Extract scenarios into .minecraft/config/pmweather/simulations/

__HEAVILY RECOMMENDED:__ Allocate atleast 4GB of RAM in launcher, having less can result in OOM crashes!
